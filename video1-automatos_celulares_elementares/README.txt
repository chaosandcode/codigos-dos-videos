Classificação feita utilizando 12 tipos de padrões encontrados dentre os apresentados pelas 256 regras dos autômatos celulares elementares.

Versão 1: 
Descrição: Gera um autômato celular para uma regra específica aplicada por um determinado número de passos de tempo.

Versão 2: 
Descrição: Gera um autômato celular para cada uma das 256 regras aplicadas por um determinado número de passos de tempo.
